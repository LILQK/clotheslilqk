# clotheslilqk
Is clothesmerfik but commands
you can use:
/camiseta - take off t-shirt
/pantalones - take off pants
/zapatos - take of shoes
/ropa - put on your clothes

---- SPANISH: -----

/camiseta - quitarte la camiseta
/pantalones - quitarte los patalones
